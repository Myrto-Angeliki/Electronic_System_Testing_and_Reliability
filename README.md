# Electronic_System_Testing_and_Reliability

Each folder contains a project that is based on one of lab projects that I had to complete for my Electronic System Testing and Reliability course. These projects are:

- **Lab1**: Design of a *Testable-Ready* circuit (**TRCUT**) by adding a scan chain to a **Circuit under Test**.
<br><br>
- **Lab2**: Validation of the **TRCUT** by inserting the values produced by an 8-bit **LFSR** into the scan chain and feeding the output of the scan chain into a **MISR**, to produce a signature that validates the correct operation of the **CUT**.
 <br><br>

---

## Requirements

- **ModelSim or other HDL simulation software**
- **Verilog (all versions) or SystemVerilog**
